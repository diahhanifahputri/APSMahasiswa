Tutorial Membuat CRUD di CodeIgniter

Assalamu'alaikum Warahmatullahi Wabarakatuh..

Pada pertemuan kali ini saya akan membahas tentang bagaimana cara membuat CRUD di codeigniter. Nah, sebelumnya saya sudah membahas tentang Implementasi MVC pada CodeIgniter dan sekarang saya akan bahas cara mengimplementasikannya saat membuat CRUD. 
Lalu apa itu CRUD? CRUD, merupakan singkatan dari Create, Read, Update dan Delete . Berikut penjelasannya:
  • C (Create): yang berarti membuat sebuah data baru, contoh kita sedang melakukan registrasi disebuah web itu sudah merupakan Create dari CRUD karena kita membuat dan menyimpan data registrasi ke database.
  • R (Read): Membaca atau menampilkan suatu data yang tadinya berada didatabase MySQL misalnya, kemudian ditampilkan di WEB menggunakan bahasa pemrograman Php
  • U (Update): nah untuk yang satu ini prosesnya adalah mengedit sebuah data dari database yang kemudian di edit menggunakan bahasa pemrograman Php berupa WEB. Contoh edit profil facebook.
  • D (Delete): Pastinya Anda tahu fungsinya apa bukan? Fungsinya hampir sama dengan Update akan tetapi proses ini adalah untuk melakukan penghapusan data di database melalui bahasa Php. Contoh pada sebuah blog terkadang ada komentar, kemudian kita hapus komentar tersebut, nah itu sudah termasuk proses delete dalam CRUD.
Perhatikan langkah-langkah di bawah ini.

A. Menampilkan Database

  1. Buka web browser lalu buatlah database xampp "http://localhost/phpmyadmin/" dengan nama db_mahasiswa dan didalamnya buatlah tabel nilai dan mahasiswa. Lihat gambar berikut ini untuk pengisian tabelnya :

  2. Setelah Database dibuat, kemudian kita lakukan konfigurasi database lewat CodeIgniter xampp/htdocs/APSMahasiswa/CodeIgniter/application/config/autoload.php. Lalu tambahkan seperti gambar dibawah ini:
  3. Lalu buka database.php atur sesuai dengan database Anda. Setelah melakukan konfigurasi, maka lanjutkan dengan membuat mvcnya.

B. Membuat Model

Langsung saja kita mulai, buka folder "CodeIgniter/application/models". Pada folder models buat file dengan nama M_Mhs.php untuk model mahasiswa(tabel mahasiswa). Kemudian isikan syntax berikut. 



$this->db->select('mahasiswa.*, nilai.IPK');
#digunakan untuk melakukan select ke dua table pada databse yang telah kita buat

$this->db->from('mahasiswa');
#digunakan untuk memilih tb_mahasiswa untuk dihubungkan dengan tb_jurusan

$this->db->join(‘nilai’, ‘nilai.id_nilai = mahasiswa.id_nilai’);
#untuk menggabungkan data dari tabel "nilai" dengan id_nilai pada tabel "mahasiswa"

$query = $this->db->get(); 
#untuk ditampilkan ke browser karena data udah siap untuk dipanggil

  2. Setelah itu pindahkan file js dan CSS yang di ekstrak ke dalam folder assetss, jika belum ada buatlah folder tersebut. buat dalam folder CodeIgniter (xampp/htdocs/APSMahasiswa/CodeIgniter/assets.)
  3. Untuk isi dari kontennya kita buat file baru di dalama folder CodeIgniter/aplication/views (letakkan diluar folder _isi) dengan nama v_tampil.php lalu isikan syntax berikut.
